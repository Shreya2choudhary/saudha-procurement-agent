# Saudha — Kirana Procurement Agent (NAM prototype)

A conversational (WhatsApp-style) procurement agent prototype for Kirana stores.

**Live demo:** https://shreya2choudhary.github.io/saudha-procurement-agent/

Single self-contained `index.html` — no build step. Features: proactive reorder agent,
multi-supplier price discovery + composite scoring, trade-scheme capture, price intelligence,
udhaar/payment tracker, single split-payment checkout, and order tracking.
